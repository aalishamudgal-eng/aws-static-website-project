# AWS Static Website Hosting Project

## 📌 Project Overview
This project demonstrates how to host a static website using AWS S3.

## 🚀 Services Used
- Amazon S3 (for hosting)
- IAM (for access control)
- CloudWatch (for monitoring - basic understanding)

## 🛠️ Steps Performed
1. Created an S3 bucket
2. Enabled static website hosting
3. Uploaded HTML and CSS files
4. Set bucket permissions to public
5. Accessed the website using S3 endpoint URL

## 🎯 My Role
- Configured S3 bucket
- Uploaded website files
- Managed permissions
- Tested website functionality

## 📚 Learning Outcome
- Learned how to host a website on AWS
- Understood S3 storage and permissions
- Gained hands-on cloud deployment experience
